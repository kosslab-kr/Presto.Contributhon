### 체크리스트

퍼블리싱 체크리스트 작성하기

1. 반응형이 된다.
2. html validation error 및 warning 체크하고 적용하기
3. 디자인 디테일 점검하기
4. 전체적인 html-css-sass 컨벤션 및 리팩토링 체크한다.

### 1) 반응형이 된다.

- 미디어쿼리 및 min-width 를 적절히 적용시켜서 수정해보기 !

[MarkupValidationCheck](https://validator.w3.org/)

### 4) 전체적인 html-css-sass 컨벤션 및 리팩토링 체크한다.
